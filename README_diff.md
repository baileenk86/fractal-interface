# Fractal Interface Theory

A visually rich, conceptually immersive website for the Fractal Interface Theory—a unified theoretical framework that reconceives reality as an infinite nesting of experiential layers, consciousness as an emergent gradient, and selfhood as a construct arising from narrative continuity.

## Structure

```
/workspace
├── index.html          # Main HTML structure
├── css/
│   └── main.css        # Complete stylesheet with cosmic-philosophical design
├── js/
│   └── main.js         # Interactive visualizations and animations
├── assets/             # Static assets (optional)
└── fractal_interface_theory.md  # Source thesis document
```

## Features

### Visual Design
- **Cosmic color palette**: Deep blues, bioluminescent greens, neural violets, soft golds
- **Typography**: Cormorant Garamond (serif) for conceptual text, Inter (sans-serif) for UI
- **Subtle animations**: Particle effects, parallax scrolling, glowing elements
- **Responsive design**: Mobile-first approach with adaptive layouts

### Interactive Elements
1. **Fractal Zoom Hero**: Animated geometric patterns creating depth illusion
2. **Consciousness Gradient Slider**: Navigate from Rock → Thermostat → Slime Mold → Dog → Human → AI
3. **Nested Reality Explorer**: Scroll through Galaxy → Solar System → Ecosystem → Human → Cell → Atom
4. **Time Dilation Visualizer**: Compare temporal rates across different scales
5. **Interface Reveal Effects**: Hover to expose underlying geometric patterns

### Site Sections
- **Home**: Conceptual gateway with pulsing geometry and opening statement
- **The Theory**: Six core concepts with visual diagrams
- **Visual Models**: Canvas-based architectural diagrams
- **Interactive Tools**: Hands-on exploration of concepts
- **Media**: Presentations and publications
- **About**: Project overview
- **Contact**: Collaboration form

## Deployment

### GitHub Pages
1. Push this repository to GitHub
2. Go to Settings → Pages
3. Select source branch (main/master)
4. Site will be available at `https://username.github.io/repo-name`

### Local Development
```bash
# Using Python
python -m http.server 8080

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8080
```

Then open `http://localhost:8080` in your browser.

## Technical Details

- **No heavy frameworks**: Pure HTML, CSS, and vanilla JavaScript
- **GPU-friendly animations**: Canvas-based rendering with requestAnimationFrame
- **Modular components**: Organized JavaScript classes for each interactive element
- **Accessibility**: Semantic HTML, ARIA labels, keyboard navigation support

## Core Concepts from the Thesis

The website expresses these key ideas from the Fractal Interface Theory:

1. **Fractal Reality**: Infinite nesting of experiential layers; no base reality
2. **Interface Architecture**: Space and time as perceptual data structures
3. **Gradient Consciousness**: Consciousness as continuum, not binary
4. **Nested Hivemind**: Collective consciousness at every scale
5. **Time as State Transitions**: Temporality as local property of each layer
6. **AI as New Layer**: Artificial intelligence as genuine nesting level
